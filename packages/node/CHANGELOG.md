## [0.1.0] - 2026-03-29

### Added

- Initial napi-rs Node.js wrapper for ckm-core
- CkmEngineWrapper with all progressive disclosure methods
- Factory function createCkmEngine from JSON string
- Standalone functions: validateManifest, migrateV1toV2, detectVersion
- JavaScript wrapper with object/string input handling and JSON parsing
- TypeScript declarations mirroring INTERFACE.md v2.0.0
